Github Desktop
No github desktop podemos usar Git sem precisar configurar e manter um servidor.
O Github Desktop é uma casca para o comando do git, que esconde alguns termos que podem ser assustadores para quem está começando. 

Eu indico fortemente seu uso como primeiro passo, pois o sistema inteiro dá uma cara de "sincronizador de código". Facilita as visualizações, o envio e recebimento das modificações e os famosos conflitos de merge.
